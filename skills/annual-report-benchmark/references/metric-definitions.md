## 0. 统一口径

- 仅以 `references/required-fields.md` 为唯一字段字典。
- 任何计算项都必须回填到同名字段，不得新增别名字段。
- 人均薪酬计算禁止使用"应付职工薪酬"余额替代"应付职工薪酬本期减少合计金额"。

## 1. 数据质量标记

- 披露值：年报直接披露
- 计算值：按公开字段计算
- 平均余额法估算：未披露原指标，按平均余额法估算
- 跨年报补齐：`Y` 年报缺对比期值，从 `Y-1` 年报补提
- 缺失：关键字段不足，无法计算
- 未披露：年报仅定性描述或未给数值

## 2. 公式

- 现金储备 = 货币资金 + 交易性金融资产 + 应收票据
- 毛利率 = (营业收入 - 营业成本) / 营业收入
- 净利率 = 归属于母公司股东的净利润 / 营业收入
- 存货周转率（估算） = 营业成本 / 平均存货
- 应收账款周转率（估算） = 营业收入 / 平均应收账款
- 人均营收 = 营业收入 / 报告期末在职员工的数量合计
- 人均毛利 = (营业收入 - 营业成本) / 报告期末在职员工的数量合计
- 人均净利 = 归属于母公司股东的净利润 / 报告期末在职员工的数量合计
- 人均薪酬（支付口径） = 应付职工薪酬本期减少合计金额 / 报告期末在职员工的数量合计

其中：

- 平均存货 = (期末存货 + 上期末存货) / 2
- 平均应收账款 = (期末应收账款 + 上期末应收账款) / 2

## 3. 同比

- 优先使用年报披露同比。
- 未披露同比时：同比 = (`Y年值` - `Y-1年值`) / abs(`Y-1年值`)
- `Y-1年值` 为 0 或缺失时，同比置空并标记“缺失”。

## 4. 估算备注

- 凡“平均余额法估算”行，备注必须写明：
  - 估算值
  - 受披露口径和期末结构影响，精度有限

# 中国金融工具集

一套专为中国金融数据分析设计的 Claude Code 插件集合。

## 插件列表

| 插件                               | 说明                                             |
| ---------------------------------- | ------------------------------------------------ |
| [cninfo-mcp](./plugins/cninfo-mcp) | 通过 MCP 协议从巨潮资讯查询和下载A股上市公司年报 |

## 安装方式

将本 Marketplace 添加到 Claude Code：

```
/plugin marketplace add youhaozhao/china-fin-toolkit
```

然后安装插件：

```
/plugin install cninfo-mcp
```
